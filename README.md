# GdBrowse
A simple but convenient browser. Use it if you don't want to download a browser.

[ENG]
The browser is built in Python.
HERE ARE THE PIP LIBRARIES USED IN THE PROJECT:
py -m pip install PyQt6 PyQt6-WebEngine pyinstaller
